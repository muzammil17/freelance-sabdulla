*******INSTRUCTIONS FOR RUNNING THE PROJECT*********

1. CHECK YOUR NODE VERSION IT SHOULD BE NODE v16.19.1
2. PLEASE INSTALL YARN CLI TO INSTALL THE DEPENDENCIES
3. CLONE THE PROJECT USING CMND: git clone https://<YOUR GITHUB TOKEN>@github.com/muzammil17/freelance-sabdulla.git
4: SWICH BRANCH TO DEV CMND: git checkout dev
5. INSTALL DEPENDENCIES USING CMND: yarn
6. TO RUN THE PROJECT USE THIS CMND: yarn serve

7. FOR BUILD THE PROJECT USE: yarn build

NOTE: YOU CAN CHANGE THE BASE URL FOR PRODUCTION AND STAGING IN CONSTANT FILE IN src/constants/index.js

THATS IT!
